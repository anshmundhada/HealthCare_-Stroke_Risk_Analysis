# HealthCare_Stroke_Risk_Analysis
🏥 Healthcare Stroke Risk Analysis Dashboard | Power BI

📌 Project Overview

This project presents an interactive Healthcare Stroke Risk Analysis Dashboard built in Power BI to analyze stroke risk patterns using patient demographic, lifestyle, and medical data. The dashboard transforms raw healthcare data into meaningful insights, helping identify high-risk groups and supporting data-driven healthcare decisions.

🎯 Objectives

* Analyze stroke cases across different age groups.
* Identify the impact of BMI, glucose levels, hypertension, and heart disease on stroke risk.
* Examine lifestyle factors such as smoking status, work type, marital status, and residence type.
* Build an interactive dashboard for effective healthcare analytics.

📊 Dashboard Highlights

* Total Patients, Stroke Cases, Stroke Rate & Average BMI KPIs
* Stroke Analysis by Age Group
* Stroke Cases by BMI Category
* Stroke Cases by Glucose Category
* Average Glucose vs Age Analysis
* Average Glucose vs BMI Analysis
* Gender Distribution
* Residence Type Distribution
* Smoking Status Analysis
* Work Type Analysis
* Hypertension & Heart Disease Impact
* Health Risk Level Distribution
* Interactive Tables & Visual Insights

🔍 Key Insights

* Senior Citizens have the highest number of stroke cases.
* Stroke risk increases significantly with hypertension and heart disease.
* Obese and overweight patients contribute to a larger share of stroke cases.
* Patients with elevated glucose levels show higher stroke risk.
* Smoking habits and work type also influence stroke occurrence.
* The overall stroke rate in the dataset is approximately 4.87%.
